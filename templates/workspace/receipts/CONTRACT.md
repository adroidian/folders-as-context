# CONTRACT.md — receipts/

## Purpose
Evidence artifacts: command output, logs, screenshots, verification notes, and audit results.

## Agent behavior
- Preserve exact useful outputs.
- Redact secrets.
- Prefer receipts over memory when resolving factual disputes.

## Authority level
Evidence.

## Movement rules
Receipts may be archived when inactive but remain evidence.

## Hard stops
Do not edit receipts except to redact secrets or correct metadata with a note.
